iPhone：Safari 下方分享 →「加入主畫面」

畫面下方五個按鈕（首頁 / 提醒 / 庫存 / 衣櫥 / 百科），隨時切換，不需返回上一頁。

3
📋 提醒系統
帳單、訂閱、家務、健康，週期性事項全管理
類型適合記錄
💰 帳單費用水電費、房租、保費、電話費
📱 訂閱服務Netflix、Spotify、Apple iCloud
🧴 消耗品定期補貨提醒
🧹 家務洗冷氣、換濾水器、清冰箱
💊 健康醫療健康檢查、預防針、回診
📌 其他不屬於以上的事項
1
新增提醒
點右下角 ＋，選類型、填名稱、設到期日與週期（共 9 種：單次 → 每年）。
2
標為完成 → 自動產生下次
點卡片 → 詳情 →「標為完成」。有金額的會先詢問本次實際金額（可修改）。完成後自動新增下一期。
3
分組顯示，可展開 / 收折
分為「逾期 / 今天 / 7天內 / 其他 / 已完成」，點標題可切換。即將到期預設展開，其他預設收折。
🔄 自動計算下次範例
水電費「每月 10 日」，5 月 10 日完成 → 自動新增「6 月 10 日」，不需手動建立。
📊 GSheet 格式
工作表名含 reminder，欄位：title, type, recurrence, dueDate, amount, description
4
📦 庫存管理
家中物品數量追蹤，快速加減不需進詳情
1
新增品項
點右下角 ＋，選分類、填名稱數量。
警告值：0 = 用完才提醒；2 = 剩 2 個以下就標紅。
2
快速加減數量
在品項卡片上直接點 − ＋，不需進詳情頁。
3
切換顯示 + 排序
右上角 ⊞ 網格卡片 / ☰ 列表，列表顯示存放位置。排序選單：名稱（預設）/ 新增日期。
4
待買清單
切到「🛒 待買」頁籤。庫存不足自動帶入；下方可自由新增臨時項目。
買到了點項目 → 打勾 + 刪除線；確認後點 ✕ 永久刪除。
📸 品項照片
新增品項時貼圖片網址，卡片顯示商品圖（完整顯示不裁切）。無圖自動顯示分類 Emoji。
📊 GSheet 格式
工作表名含 inventory，欄位：name, category, quantity, unit, threshold, imageUrl, barcode, location, note
5
👗 電子衣櫥
管理孩子衣物的尺碼與換季狀態
1
新增衣物（照片欄在最上方）
點右下角 ＋。照片網址欄位在最上方，貼上後立即預覽。再填名稱、尺碼、類型等。
2
更新換季狀態
孩子穿不下了？點卡片 → ✏️ 編輯 → 改狀態「穿不下」或「已轉贈/售出」。
3
排序 + 切換模式
右上角排序選單：新增日期（預設，最新在前）/ 名稱。切換 ⊞ / ☰ 模式。
4
釘選常穿衣物
詳情頁點 📍，加入頁面頂部橫向捲動快速區，不用翻找。
📊 詳情頁雙欄排版
類型 / 尺碼、袖長 / 來源、狀態 / 已穿，三組雙欄對稱顯示，無圖時顯示 👗 Emoji。
📊 GSheet 格式
工作表名含 wardrobe，欄位：name, type, size, sleeve, source, status, location, worn, note, imageUrl，worn 填 true 或 false。
6
📖 小百科
收藏育兒資訊、食譜、好用連結
1
新增書籤
點右下角 ＋，貼網址後按「抓取預覽」，自動填入標題和封面圖。無圖自動顯示分類 Emoji。
2
加上 Markdown 筆記
書籤可附加筆記，支援 Markdown：# 標題、**粗體**、- 清單，含即時預覽。
3
排序 + 搜尋
右上角排序：新增日期（預設）/ 名稱。搜尋框支援標題、標籤、網域名稱。9 種分類 pill 快速篩選。
4
釘選重要書籤
詳情頁點 📍，出現在小百科頂部快速查找區，同步到首頁「精選書籤」。
📊 GSheet 格式
工作表名含 wiki，欄位：title, category, tags, url, imageUrl, note。tags 多個標籤用空格分隔。
7
📊 Google Sheet 批量匯入
在 Excel 填好資料，一鍵同步
⚠️ 使用前必做：公開設定
Google Sheet → 右上角「共用」→「一般存取權」→「知道連結的人」→「檢視者」→ 完成。
未設定公開，App 無法讀取資料。
1
在 Sheet 填好資料
→ 點此開啟家庭 Sheet 模板
第 1 列填欄位標題，從第 2 列開始填資料。
2
進入對應的 App 模組
要匯入庫存 → 進「📦 庫存管理」；要匯入衣物 → 進「👗 電子衣櫥」，依此類推。
3
點右上角 📥 → 點「📊 從 Google Sheet 直接匯入」
確認後開始同步，完成顯示「✅ 成功匯入 N 筆」。
每次匯入為新增，不覆蓋現有資料。
模組工作表頁籤（名稱含此字即可）欄位數
📦 庫存inventory9 欄（A–I）
👗 衣櫥wardrobe10 欄（A–J）
📋 提醒reminder6 欄（A–F）
📖 百科wiki6 欄（A–F）

❌ 匯入失敗常見原因
1. Sheet 未設「知道連結的人可以檢視」
2. 工作表名不含對應英文字
3. 第 1 列欄位標題有錯字或多空格
4. 錯誤訊息會顯示偵測到哪些工作表，方便對照
8
❓ 常見問題
資料沒有即時更新
確認已連網，下拉重新整理。仍不更新則關閉頁面重新開啟。
忘記房間碼
詢問家中其他已登入的成員，或聯絡 App 管理者。
天氣沒有顯示
確認瀏覽器允許「位置存取」。iPhone：設定 → Safari → 位置 → 允許。
書籤抓取預覽失敗
手動填入標題即可，封面圖可自行貼圖片網址。
Google Sheet 匯入失敗
先確認 Sheet 已設公開；確認工作表名含對應英文字；看錯誤訊息裡顯示偵測到哪些工作表名稱。百科支援中英文欄位標題（title / 標題 / 書籤 / 名稱 均可）。
不小心刪錯資料
刪除後無法還原，請謹慎操作。重要資料建議用 Google Sheet 備份。
可以在電腦上用嗎
可以！在電腦瀏覽器開啟同一網址，輸入房間碼即可，完整支援電腦版。
