📦 庫存管理
家中物品追蹤，快速加減不需進詳情
1
新增品項
右下角 ＋，選分類、填名稱數量。
警告值：0 = 用完才提醒；2 = 剩 2 個以下標紅警告。
2
快速加減數量
卡片上直接點 − ＋，不需要進詳情頁。
3
切換模式 + 排序
右上角 ⊞ 網格 / ☰ 列表（顯示存放位置）。排序：名稱（預設）/ 新增日期。
4
分類 Accordion 預設收折
點分類標題展開或收折。低庫存的分類顯示 ⚠️。
📸 品項照片
填入圖片網址，卡片顯示商品圖（contain 完整顯示不裁切）。無圖自動顯示分類 Emoji。
5
🛒 待買清單
在庫存管理頁切換到「待買」頁籤

📦 庫存不足自動加入

✓
逛超市邊買邊打勾
點項目 → 打勾＋刪除線，清楚標示已買。狀態存手機本地，重整不消失。
✕
✕ 單筆移除
點右側 ✕，從待買清單暫時移除（不影響庫存數量）。
🗑
一鍵清除已勾選
有打勾時，頂部出現「✓ 清除已勾選項目」按鈕，結帳後一次清乾淨。
買到了打勾 ✓
→
回家到庫存清單點 ＋ 補數量
→
數量足夠後自動消失
💡 不想追蹤某樣東西了？
例如「柔膚巾不需要再買」→ 去庫存清單把警告值改成 0（用完才提醒）或直接刪除品項。不要只在待買清單刪除，那樣下次數量變動還是會再出現。

✏️ 自由新增區（非庫存品項）

操作說明
點項目打勾＋刪除線
點 ✕永久刪除這筆
點「清除已完成」一次刪掉所有已打勾項目
💾 自由填寫區存在手機本地，重整不消失。

👗 電子衣櫥
管理孩子衣物的尺碼與換季狀態
1
新增衣物
右下角 ＋。照片網址欄在最上方，貼上即時預覽。填名稱、尺碼、類型、來源、狀態。
2
更新換季狀態
點卡片 → ✏️ 編輯 → 改狀態（當季 / 非當季 / 穿不下 / 已轉贈）。頂部四格統計即時更新。
3
排序 + 切換模式
排序：新增日期（預設，最新在前）/ 名稱。⊞ 網格 / ☰ 列表切換。
4
📌 釘選常穿衣物
進詳情頁點 📍，加入頂部橫向快速區，常穿的不用翻找。
🖼 詳情頁
雙欄對稱排版。圖片 contain 完整顯示，無圖顯示 👗 Emoji。
7
📖 小百科
收藏育兒資訊、食譜、好用連結
1
新增書籤
右下角 ＋，貼網址 →「抓取預覽」自動填入標題和封面圖。無圖自動顯示分類 Emoji。
2
附加 Markdown 筆記
書籤可加筆記：# 標題 **粗體** - 清單，含即時預覽。
3
排序 + 搜尋 + 分類篩選
排序：新增日期（預設）/ 名稱。搜尋標題、標籤、網域。9 種分類 pill 快速篩選。
4
📌 釘選書籤 → 同步到首頁
進詳情頁點 📍 釘選。釘選的書籤出現在小百科頂部快速區，同時自動同步到首頁「精選書籤」。
📌 釘選書籤流程
小百科 → 書籤詳情 → 點 📍
→
首頁「精選書籤」自動出現

📋 CSV 批量匯入
在電腦整理好資料，貼上即匯入
1
用 Excel / Numbers 整理資料
照下方格式，第 1 列填欄位標題，從第 2 列填資料。複製全部儲存格。
2
進入對應模組 → 點右上角 📥
要匯入庫存進庫存管理，要匯入百科進小百科，依此類推。
3
貼上 CSV 文字 → 點「匯入」
完成顯示「✅ 成功匯入 N 筆」。每次匯入為新增，不覆蓋現有資料。

📦 庫存格式

name,category,quantity,unit,threshold,imageUrl,barcode,location,note
雞蛋,food,10,顆,0,,,冰箱,
衛生紙,clean,3,包,2,,,浴室,抽取式
嬰兒濕巾,baby,5,包,2,,,客廳,敏感膚質
fooddrinkfrozencleanbathmedicalbabykitchenofficeother

threshold：0 = 用完才提醒，>0 = 低於此值提醒

❓ 常見問題
資料沒有即時更新
確認已連網，下拉重新整理。仍不更新關閉重開。
天氣沒有顯示
確認瀏覽器允許位置存取。iPhone：設定 → Safari → 位置 → 允許。
書籤抓取預覽失敗
手動填入標題即可，封面圖可另外貼圖片網址。
首頁沒有顯示精選書籤
進小百科，點任一書籤 → 詳情 → 點右上角 📍 釘選。釘選後首頁會自動出現。
待買清單打勾後重整消失
打勾狀態存在手機本地。若消失，確認是否清除了瀏覽器快取。
某樣庫存品不想追蹤了
進庫存清單點卡片 → ✏️ → 把警告值改成 0，或直接刪除品項。
刪錯資料了
刪除後無法還原，建議重要資料定期用 CSV 匯出備份。
可以在電腦上用嗎
可以，開啟同一網址輸入房間碼即可，完整支援電腦瀏覽器。
