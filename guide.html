🚀 開始使用
第一次登入只需要輸入家庭房間碼
1
打開 App 網址
在手機瀏覽器（Safari 或 Chrome）開啟：
yui0151.github.io/home-hub/
2
輸入家庭房間碼
全家人共用同一組房間碼，輸入後點「進入我的家庭」。登入後不需要重複輸入。
3
加到手機桌面（推薦）
iPhone：Safari → 分享按鈕 →「加入主畫面」
Android：Chrome → 右上選單 →「加到主畫面」
之後像 App 一樣一鍵開啟。
✅ 即時同步
爸爸在超市更新庫存，媽媽的手機馬上看到。所有資料存在雲端，換手機也不怕。
📱 底部導覽列
畫面下方五個按鈕（首頁 / 提醒 / 庫存 / 衣櫥 / 百科），任何時候都可以切換，不需要返回上一頁。

📌 快速記錄專區（首頁最實用功能）

📝
備忘錄
隨手記下任何事，自動儲存，不需按儲存鍵。Tab 旁無數字時代表沒有內容。
✅
待辦事項
打勾完成，點 ✕ 刪除。Tab 旁的數字 = 未完成項目數量，一眼看還有幾件事沒做。
🛍️
待買清單
快速記下要買的東西，Tab 旁顯示未購買數量。這裡是個人隨手清單，庫存不足的會另外在庫存模組管理。
3
📋 提醒系統
帳單、訂閱、家務、健康，週期性事項都在這裡管
類型適合記錄什麼
💰 帳單費用水電費、房租、保費、電話費
📱 訂閱服務Netflix、Spotify、Apple iCloud
🧴 消耗品定期補貨提醒（尿布、牛奶等）
🧹 家務洗冷氣、換濾水器、清冰箱
💊 健康醫療健康檢查、預防針、回診時間
📌 其他不屬於以上的事項
1
新增提醒
點右下角 ＋，選類型、輸入名稱、設定到期日與週期（9 種：單次 → 每年）。
2
標為完成 → 自動產生下次
點卡片進詳情，按「標為完成」。若有金額，會先問本次實際金額（可修改，帳單金額每期可能不同）。完成後，系統自動新增下一期的提醒。
3
分組顯示，可收折
分為「逾期 / 今天 / 7天內 / 其他 / 已完成」，點標題可展開或收折。預設：即將到期展開，其他與已完成收折。
🔄 自動計算下次範例
水電費設定「每月 10 日」，5月10日標完成 → 系統自動新增「6月10日」的提醒。 每次完成都可以記錄當期實際金額，歷史紀錄完整保存。
📊 Google Sheet 匯入提醒
Sheet 工作表頁籤名稱需含「reminder」，欄位：title, type, recurrence, dueDate, amount, description
4
📦 庫存管理
家中所有物品的數量追蹤，快速加減不需進詳情
1
新增品項
點右下角 ＋，選分類、填名稱、數量。
警告值：設 0 = 用完才提醒；設 2 = 剩 2 個以下就標紅色提醒。
2
快速加減數量
在品項卡片右側直接點 − ＋，不需要進詳情頁。
3
切換顯示模式
右上角 ⊞ 卡片 / ☰ 列表切換。列表模式會額外顯示存放位置，找東西更快。
4
待買清單
切到「待買清單」頁籤。上方自動列出庫存不足的品項；下方可自由新增臨時要買的東西（如：蛋糕、禮物）。買到了打勾，確認沒問題後點 ✕ 刪除。
📸 品項照片
新增品項時貼上圖片網址，卡片就會顯示商品圖（完整呈現，不裁切）。無圖時自動顯示分類 Emoji。
📊 Google Sheet 匯入庫存
Sheet 工作表頁籤名稱需含「inventory」，欄位：name, category, quantity, unit, threshold, imageUrl, barcode, location, note
5
👗 電子衣櫥
管理孩子衣物的尺碼與換季狀態
1
新增衣物（照片欄在最上方）
點右下角 ＋。照片網址欄位在最上方，貼上後馬上預覽。再填名稱、尺碼、類型、來源、狀態。
2
更新換季狀態
孩子穿不下了？點卡片 → ✏️ 編輯 → 改狀態為「穿不下」或「已轉贈/售出」。統計卡會即時更新。
3
釘選常穿衣物
進詳情，點右上角 📍 釘選，最常穿的衣服出現在頁面頂部橫向捲動快速區，不用翻找。
📊 四格統計卡
頁面頂部顯示：總計 / 當季 / 非當季 / 未穿過，換季整理時一目了然。
🔍 雙重篩選
可同時篩選「狀態」（當季 / 非當季…）和「類型」（外套 / 上衣…），快速縮小查找範圍。
📊 Google Sheet 匯入衣櫥
Sheet 工作表頁籤名稱需含「wardrobe」，欄位：name, type, size, sleeve, source, status, location, worn, note, imageUrl
6
📖 小百科
收藏育兒資訊、食譜、好用連結
1
新增書籤
點右下角 ＋，貼上網址後按「抓取預覽」，系統自動填入標題和封面圖。無圖時顯示分類 Emoji，不留白。
2
加上 Markdown 筆記
書籤可附加筆記，支援 Markdown 格式：# 標題、**粗體**、- 清單，含即時預覽。
3
釘選重要書籤
進詳情，點 📍 釘選。釘選書籤出現在小百科頂部快速查找區，同時同步到首頁「精選書籤」。
4
搜尋與切換模式
搜尋框支援標題、標籤、網域名稱。右上角 ⊞ 網格 / ☰ 列表切換，9 種分類 pill 快速篩選。
📊 Google Sheet 匯入百科
Sheet 工作表頁籤名稱需含「wiki」，欄位：title, category, tags, url, imageUrl, note。tags 多個標籤以空格分隔。
7
📊 Google Sheet 批量匯入
在 Excel 填好資料，一鍵同步到 App
⚠️ 使用前必須完成：公開設定
Google Sheet → 右上角「共用」→「一般存取權」→ 改成「知道連結的人」→「檢視者」→ 完成。
未設定公開時，App 無法讀取資料。
1
在 Google Sheet 填好資料
→ 點此開啟家庭 Sheet 模板
第 1 列填英文欄位標題，從第 2 列開始填資料。
2
打開對應的 App 模組
要匯入庫存 → 進「📦 庫存管理」；要匯入衣物 → 進「👗 電子衣櫥」，依此類推。
3
點右上角 📥 → 點「📊 從 Google Sheet 直接匯入」
確認對話框後開始同步，完成顯示「✅ 成功匯入 N 筆」。
注意：每次匯入為新增，不會覆蓋現有資料。

❌ 匯入失敗常見原因
1. Sheet 沒設定「知道連結的人可以檢視」
2. 工作表頁籤名稱不含對應英文字（大小寫不限）
3. 第 1 列欄位名稱有錯字或多餘空格
4. 網路不穩，重試一次即可
8
❓ 常見問題
遇到問題先看這裡
資料沒有即時更新
確認手機已連網，下拉重新整理頁面。仍不更新則關閉頁面重新開啟。
忘記房間碼了
詢問家中其他已登入的成員，或聯絡 App 管理者。
天氣沒有顯示
確認瀏覽器允許「位置存取」。iPhone 設定：設定 → Safari → 位置 → 允許。
書籤抓取預覽失敗
手動填入標題即可，封面圖可自行貼圖片網址覆蓋。
Google Sheet 匯入顯示「❌ 匯入失敗」
先確認 Sheet 已設定公開檢視；再確認頁籤名稱含有對應英文字；錯誤訊息會顯示偵測到哪些工作表名稱，方便對照。
不小心刪錯資料了
目前刪除後無法還原，請謹慎操作。重要資料建議定期用 Google Sheet 備份。
可以在電腦上使用嗎
可以！在電腦瀏覽器開啟同一個網址，輸入房間碼即可，完全支援電腦版瀏覽器。
多人同時操作會衝突嗎
資料使用 Firebase Firestore 即時同步，最後一次寫入生效，建議避免同時編輯同一筆資料。
