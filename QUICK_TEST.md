## 📋 提醒系統 — 完整測試流程

### 測試 1：新增提醒

```
1. 打開 module-reminders.html
   ↓
2. 點擊浮動按鈕 ＋（右下角）
   ↓
3. 填入資料：
   - 提醒名稱：「水電費」
   - 類型：「財務」
   - 週期：「每月」
   - 到期日期：「2026-05-12」
   - 金額：「1200」
   ↓
4. 點「儲存」按鈕
   ↓
5. ✅ 檢查清單中是否出現新提醒
```

**預期結果：**
- Modal 關閉
- 新提醒立即出現在列表
- Firebase 自動保存（無需任何操作）

### 測試 2：編輯提醒

```
1. 點擊任何提醒卡片（「水電費」）
   ↓
2. Modal 打開，顯示所有資料
   ↓
3. 修改資料：
   - 金額改為「1250」
   ↓
4. 點「儲存」
   ↓
5. ✅ 清單中金額即時更新
```

### 測試 3：標記完成

```
1. 在提醒卡片右側找到圓圈 ○
   ↓
2. 點一下圓圈
   ↓
3. ✅ 變成綠色 ✓（已完成）
   ↓
4. 再點一下
   ↓
5. ✅ 變回圓圈 ○（未完成）
```

### 測試 4：刪除提醒

```
1. 點擊提醒卡片打開編輯
   ↓
2. 會看到「刪除」按鈕（紅色）
   ↓
3. 點「刪除」
   ↓
4. 確認刪除
   ↓
5. ✅ 提醒從清單消失
```

### 測試 5：重整後資料是否存在（最重要！）

```
1. 新增一個提醒：「買牛奶」
   ↓
2. 刷新頁面（按 F5 或 Ctrl+R）
   ↓
3. ⏳ 等待 1-2 秒（Firebase 加載）
   ↓
4. ✅ 「買牛奶」提醒還在清單中
   ↓
5. 🎉 這代表資料已持久化到 Firebase！
```

## 📦 庫存管理 — 完整測試流程

### 測試 1：新增品項

```
1. 打開 module-inventory.html
   ↓
2. 點擊浮動按鈕 ＋
   ↓
3. 填入資料：
   - 品項名稱：「白米」
   - 分類：「食品」
   - 數量：「5」
   - 單位：「kg」
   - 警告值：「2」
   - 位置：「廚房」
   ↓
4. 點「儲存」
   ↓
5. ✅ 品項出現在食品分類下
```

### 測試 2：快速增減數量

```
1. 找到「白米」卡片
   ↓
2. 右側有數字「5」和 ± 按鈕
   ↓
3. 點「＋」一次
   ↓
4. ✅ 數字變成「6」（即時更新）
   ↓
5. 點「－」兩次
   ↓
6. ✅ 數字變成「4」
   ↓
7. 再點「－」兩次
   ↓
8. ✅ 數字變成「2」（等於警告值）
   ↓
9. 再點「－」一次
   ↓
10. ✅ 卡片變紅色（警告！低庫存）
```

### 測試 3：編輯品項

```
   ↓

   ↓
4. 點「儲存」
   ↓
5. ✅ 卡片下方位置立即更新為「冰箱」
```

### 測試 4：重整後資料是否存在

```
   ↓
```

## 🔄 多設備同步測試（高級）

### 測試跨設備同步

```
設備 A（電腦）          設備 B（手機）
     ↓                      ↓
打開首頁                  打開首頁
同一個房間碼 MY0918       同一個房間碼 MY0918
     ↓                      ↓
在提醒系統               等待...
新增「買菜」
點儲存
     ↓
即時上傳到 Firebase
     ↓                      ↓
                   Firebase 通知設備 B
                   自動刷新清單
                   立即顯示「買菜」✓
```

**驗證步驟：**
1. 用電腦打開一個視窗（module-reminders.html）
2. 用手機打開另一個視窗（同一個 URL + 房間碼）
3. 在電腦上新增一個提醒
4. **不刷新手機頁面**
5. ✅ 手機應立即顯示新提醒（1-2 秒內）

## 🐛 除錯提示

### 如果資料沒有保存

```
1. 檢查網路連線是否正常
2. 打開 DevTools Console
3. 查看是否有錯誤信息
4. 確認 Firebase 配置是否正確
5. 確認房間碼是否一致
```

### 如果 Modal 不打開

```
1. 檢查是否點擊了正確的按鈕
2. F12 打開 DevTools
3. 檢查 Console 中是否有 JavaScript 錯誤
4. 嘗試重整頁面
```

### 如果多設備不同步

```
1. 確認使用了完全相同的房間碼
2. 確認都連接到網路
3. 等待 1-2 秒
4. 手動刷新看是否有資料
5. 查看 Firebase 連線狀態（⚙️ 設定）
```
